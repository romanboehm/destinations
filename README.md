# Destinations

_Destinations_ is a script to calculate, for a given set of locations, or starting points, the set of locations reachable via a direct Deutsche Bahn (DB) train connection.

## Run

### Prerequisites
- [JBang](https://www.jbang.dev/download/)

### Example
`jbang destinations.java --max-duration=60 8000260 8000105` will get you all destinations directly reachable from both Frankfurt/Main Main Station and WÃ¼rzburg Main Station within 60 minutes.
